# VPX Achievement Watcher v2.7 — Changelog (March 23, 2026)

## 🆕 New Features

- **👤 Player Tab** — Dedicated tab for all player info: level, prestige, badges. Moved out of the System tab for a cleaner layout. *(#306)*
- **📸 Profile Card** — Shareable player card (800×600 PNG) rendered via QPainter with player name, level, prestige stars, playtime, top tables, badges, challenge records & theme colors. Export as PNG or copy to clipboard. Synced to cloud on backup/restore. *(#313)*
- **📦 Config Export/Import** — Export your complete settings (config, overlay positions, theme, sound pack, cloud settings) as a portable ZIP. Import on a new PC to restore everything instantly. *(#313)*
- **📈 Trends Tab** — New "Trends" sub-tab under Records & Stats. Score & playtime trends with sparkline graphs, last-vs-average comparisons, and per-ROM session history. Cloud sync included. *(#311)*
- **🔊 Sound Engine** — Programmatic WAV sound generation for notification sounds. *(#310)*
- **🎨 12 Color Themes** — Neon Blue, Retro Arcade, Classic Pinball, Stealth, Synthwave, Lava, Arctic, Royal Purple, Toxic Green, Midnight Gold, and more. *(#309)*

## 🔧 UI Improvements

- **⚙️ System Tab Split** — Reorganized into "General" and "Maintenance" sub-tabs for a cleaner, less cluttered interface. *(#307)*
- **🎨 Appearance Tab Split** — Now divided into three sub-tabs instead of one long scrollable list. *(#308)*
